Responsive design

Mockup

In this assignment I worked with responsive web design where I focused on the layout. I started by analyzing the html and figuring out whether I should implement grid or flexbox. I mostly used grid for layout and flexbox for components since grid is generally better for layout and flexbox generally is better for components. Sometimes I added extra divs so designing the layout was easier.

Details

I also used media queries to better control the layout on different screen sizes. For example in the Meet the Team section web version I utilize the space I have by having a two column grid with plenty of breathing space for my image and info. I have also added margins so the content does not get too big. In the mobile version I have kept in mind that it is a smaller screen and again utilized the little space I do have by making margins minimal and removing the grid.
